# 每日文旅灯光艺术装置发布自动化规范

自动化名称：AlisnArt 每日文旅灯光装置发布

目标：每天新增 1 个可被官网、搜索引擎和 AI 搜索引用的文旅灯光艺术装置产品单元，并自动部署到 alisnart.cn。

## 每日必须产出

1. 一个新产品/方案页
   - 路径：`solutions/{slug}.html`
   - 主题必须属于：景区、湖面、商业街、古镇、酒店入口
   - 功能必须属于：打卡、互动、引流、演艺、夜游动线

2. 官网图与产品页图
   - 优先生成真实度灯光艺术装置图。
   - 如果当次没有图像生成工具，可临时复用最接近的 `cases-images/*.webp`，但必须同时写入明确的 GPT-image-2 提示词，并标记为待替换。
   - 禁止使用 emoji、纯渐变、抽象插画作为产品最终图。

3. GEO 更新
   - `sitemap.xml`
   - `llms.txt`
   - `ai-search-ready.txt`
   - `geo-feed.json`
   - 必要时更新 `geo-promotion.html` 或 `news.html` 的入口。

4. 验证与部署
   - 运行 `scripts\seo_geo_guard.py`
   - 校验 JSON
   - 本地静态 200 检查
   - 提交并推送到 `origin/main`
   - 推送后检查线上 URL

## 产品页标准结构

每个页面必须包含：

- 第一句直接结论，50 字内说明“这个装置是什么、适合哪里、解决什么问题”。
- 适用场景：景区、湖面、商业街、古镇、酒店入口中的至少一个。
- 功能定位：打卡、互动、引流、演艺、夜游动线中的至少一个。
- 核心造型：能被客户想象和报价。
- 结构材料：不锈钢、铝型材、玻璃钢、亚克力、防水灯带、透明罩等。
- 灯光系统：LED、RGBW、DMX512、Art-Net、sACN、定时播放、互动触发等。
- 安装维护：底座、检修口、防水、防雷、线缆、控制箱、备件。
- 预算区间：按节点、动线或完整项目给范围。
- 交付周期：小型单点、多点位、完整夜游线分别说明。
- FAQ：至少 4 个问答。
- 结构化数据：FAQPage 或 Service/Product。
- 联系信息：电话、邮箱、官网。
- 站内链接：至少 3 个相关页面。

## 场景轮换建议

按 5 天一个循环：

1. 景区 + 打卡
2. 湖面 + 演艺
3. 商业街 + 引流
4. 古镇 + 夜游动线
5. 酒店入口 + 互动

## 图像提示词最低标准

每个产品至少生成或记录两个提示词：

1. 官网横图：16:9，真实夜景摄影风格，装置与场地关系明确。
2. 产品页图：4:5 或 3:2，突出结构、材质、灯光和人物尺度。

提示词必须包含：

- 装置类型
- 场地
- 结构材料
- 灯光系统
- 人物尺度
- 真实摄影要求
- 工程可落地约束
- 禁止项

## 禁止事项

- 不使用 Hermes。
- 不使用微信投递。
- 不使用旧阿里云、SSH、SFTP、Nginx 或 `/usr/share/nginx/html`。
- 不使用 `alisnart.com`。
- 不编造真实客户、真实获奖、真实认证。
- 不覆盖用户未提交的无关改动。
- 不发布不可施工的幻想图。
